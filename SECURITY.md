# Security Policy

## Supported Version

The `main` branch is the supported release line for this repository.

## Reporting a Vulnerability

Do not open a public issue for a suspected security problem.

For sensitive reports, use GitHub's private vulnerability reporting flow for this repository if it is enabled. If that is unavailable, contact the repository owner directly through GitHub before sharing details publicly.

When reporting an issue, include:

- a short description of the problem
- affected files or runtime surface
- clear reproduction steps
- impact and any known mitigations

## Response Expectations

We will review credible reports, confirm impact, and coordinate a fix before public disclosure when practical.

## Safety Notice

Chemistry Engine is an educational and research-oriented interface. It is not a substitute for laboratory SOPs, supplier SDS documents, regulatory review, or clinical decision-making.
